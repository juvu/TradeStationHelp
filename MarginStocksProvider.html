Guosen.MarginStocksProvider

    

MarginStocksProvider 类

用于访问相关账户的融券列表信息。

命名空间：guosen

属性

名称 类型 说明
Account string 筛选需要列出您想要查询列表的某个账户。默认为全部信用账号。
Count int 基于筛选条件的现有仓位的数目。
Data[index] Object 基于下标返回特定的Data。
DataID String 返回Data数据的类型。
Load bool 设置为 true，以建立与数据提供程序的连接。
MarginStock[index] Object 获取与指定索引关联的MarginStock对象。
RealTime bool 如果所提供数据是实时数据则为 true。如果不是实时数据，则为 false。
State enum 获取的数据提供程序的状态。有关可能值的列表，请参阅 DataState。
TimeZone enum 获取 TimeZone 对象。本地 = 0，交易所 = 1。请参阅 TimeZone 对象。
TryMarginStock   程序体不存在

方法

名称 返回值 说明
Clone   类的克隆实例。
CloseProvider   关闭与 DataProvider 的连接。
Contains(symbol) bool 如果指定 Symbol 包含在提供程序中，则为 true。
Equals bool 如果指定的帐户 ID 字符串包含在当前实例中，则为 true。
GetType bool 如果指定的帐户 ID 字符串包含在当前实例中，则为 true。
LoadProvider   打开与 DataProvider 的连接。

事件

名称 说明
StateChanged 当提供程序状态更改时，此事件发生。有关处理程序 args 参数返回的属性，请参阅 StateChangedEventArgs。
Updated 融券列表更新时，发生此事件。有关处理程序 args 参数返回的属性，请参阅 MarginStockUpdatedEventArgs。

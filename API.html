Guosen.AccoutProvder

    

AccoutProvder类

用于访问账户信息的字段名称字符串集合。

当从一个账户实例读取数据值时，在 Account 对象的 Value[名称] 属性中作为名称参数来使用

命名空间：guosen

属性

名称 类型 说明
C0001 C0001 程序体不存在
C0010 C0010 无法识别的字符串

方法

名称 返回值 说明
Contains(accountID) bool 如果指定的帐户 ID 字符串包含在当前实例中，则为 true。
Contains(accountID) bool 如果指定的帐户 ID 字符串包含在当前实例中，则为 true。

事件

名称 说明
Updated 账户值更新时，发生此事件。有关处理程序 args 参数返回的属性，请参阅 AccountUpdatedEventArgs。
Updated 账户值更新时，发生此事件。有关处理程序 args 参数返回的属性，请参阅 AccountUpdatedEventArgs。
